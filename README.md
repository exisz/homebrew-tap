# Homebrew Tap — gotexis

Homebrew formulae for [gotexis](https://github.com/gotexis) open source tools.

## Install

```bash
brew tap gotexis/tap
brew install lazyjira
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| [lazyjira](https://github.com/gotexis/lazyjira) | Zero-dependency CLI for Jira Cloud |
